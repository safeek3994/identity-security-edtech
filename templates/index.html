Core Concepts Demonstrated

What You'll Learn

Explore these fundamental security concepts through live, working code.

🔐
Authentication

Verify who you are using email + password. Passwords are hashed with werkzeug.security — never stored in plain text.

🛡️
Authorization

Control what you can do. Role-Based Access Control (RBAC) restricts the Admin Panel to admin users only.

🔑
Session Management

Flask-Login manages secure sessions after login. @login_required protects routes from unauthenticated access.

🗄️
Secure Storage

User credentials stored in SQLite via Flask-SQLAlchemy. Password hashes use PBKDF2-SHA256 — cracking them is computationally infeasible.

🚫
Access Control

Unauthorized users attempting to access protected routes are redirected or shown an Access Denied response with HTTP 403.

⚡
Input Validation

Server-side validation for every form field. Duplicate email detection, password length checks, and match confirmation.
